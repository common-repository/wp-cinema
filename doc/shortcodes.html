WP Cinema Shortcodes Documentation

This page will list the main shortcodes in detail soon; for now, it only has a very basic summary of the main shortcodes.


[--wpcinema_daily]

Displays the daily schedule page including the day selection bar. This shortcode normally goes on the home page.


[--wpcinema_daily_body]

Displays the daily schedule page body (called by wpcinema_daily).


[--wpcinema_daybar]

Displays the day selection bar and date heading which go along with the daily schedule. (called by wpcinema_daily)
